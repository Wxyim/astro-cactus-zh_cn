### 改动点

1、简体中文支持，主体采用思源黑字体，代码类 Fira Code

2、加入 Giscus 评论区，支持与原有主题的明暗联动切换

3、加入静态版每日一言，每日自动切换

4、博客内容页加入了浮动目录按钮

5、调节字体大小、日期格式、颜色相关、组件布局等，提升整体观感

6、Github Pages 自动化部署配置

### 安装方法

建议查看原版，这样可以了解一些根源因素帮助您理解：[Astro Cactus Theme](https://github.com/chrismwilliams/astro-theme-cactus)

确保您已经安装了所需的软件，比如 Git、Node.js、VSCode 等。

下面是如何使用此仓库的简单步骤：

1、克隆本仓库，指向自已的 Github 远程仓库

2、本地修改文件内容，比如项目名称，个人网站信息等，博客内容在 src/content 里面书写

3、推送远程仓库，在 Github 个人仓库 Settings - Pages 中配置 GitHub Pages 选项以支持自动部署静态站
